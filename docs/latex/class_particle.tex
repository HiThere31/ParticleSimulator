\doxysubsection{Detailed Description}
Representa una partícula en el simulador. 

La clase \doxylink{class_particle}{Particle} contiene información sobre la posición, velocidad, aceleración y el radio de una partícula. También incluye métodos para actualizar su estado y dibujarla en una ventana gráfica.
